Detailed Description

Description of CachedORMModel

An ORM Model class that implements caching on Find/FindAll actions and keeps the cache updated on save() actions.

This is mainly useful when the database storing the object is on a different server from the application. If the database is on the same server then the caching within the database should be sufficient.

Note:
Use of this class is identical to ORM_Model.
