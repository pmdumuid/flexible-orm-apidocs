fetch ( $  template)

Fetch the output of a template.

Essentially running a template to get the output string. For example, using SmartyTemplate the following is functionally equivalent:

 // Fetch
 echo $smarty->display( 'myTemplate' );
 
 // Dislay
 $smarty->display( 'myTemplate.tpl' );

BaseController::performAction()
