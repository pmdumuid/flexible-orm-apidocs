select ( $  property_or_condition,
$  value = null 
)

Return a subset of a collection based on a condition

May be used in two ways-

  • Selecting by property
  • Selecting by condition

Selecting by Property

This is the simplest method of selection. Provided a property name and value, select() simply searches the collection for all cases the property is exactly equal (===) to the value.

Selecting by Condition

Select when the condition (specified as a string) is true. The current element will be named $x.

Usage: // By Property: Select all clients where current == true $current_clients = $organisation->clients()->select( 'current', true );

// By Condition: Select all clients where the age property is less than 12 $child_clients = $organisation->clients()->select( '$x->age < 12' );

Note: If you want to select all elements with a property set to null, you have to use the condition method.

Returns:
ModelIterator
Parameters:
string$property_or_conditionA property name (all elements must have this property) or a condition string
mixed$valueIf using a property, this should be the value you want to select.
See also:
map()
