Detailed Description

The Configuration class is a simple class to keep all application settings in.

Ini file groups allow the values to be kept in sensible groups.

For more information see Configuration Tutorial

static __callStatic ( $  group,
$  arguments 
) [static]

Allow ini groups to be called as static methods.

Usage: 

 // Ini file:
 [my_group]
 option = 'value'

 [another_group]
 option = 'no'

 // PHP
 echo Configuration::my_group()->option; // outputs: value

 // PHP - Alternative method:
 echo Configuration::my_group( 'option' ); // outputs: value
Parameters:
string$groupThe group name (eg "my_group" in the above example)
array$argumentsOnly the first argument is used and it is used as the property name
Returns:
The requested property value
