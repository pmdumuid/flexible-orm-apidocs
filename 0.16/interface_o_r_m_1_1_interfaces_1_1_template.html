Assign a variable to a template.

To make values available in a template, they must be assigned. This method is identical to Smarty::assign() when used with 2 arguments.

fetch (   $template)

Fetch the output of a template.

Essentially running a template to get the output string. For example, using SmartyTemplate the following is functionally equivalent:

// Fetch
echo $smarty->display( 'myTemplate' );
// Dislay
$smarty->display( 'myTemplate.tpl' );
See also:
BaseController::performAction()
Parameters:
string$templateThe name of the template to fetch
Returns:
string The output of the template.
