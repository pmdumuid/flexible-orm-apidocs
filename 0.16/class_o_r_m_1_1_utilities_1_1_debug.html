Detailed Description

Simple class to help with debugging tasks.

Using this for debugging makes it easy to disable debug remarks on production servers and control how (or if) debugs are loggged)

Simple Usage

$myObject = Car::Find();
Debug::Dump($myObject);
// Would output the contents of $myObject

Options You can disable output system wide by calling Debug::SetDisplayOutput(false) . If you don't alter the 'SetDisplayOutput' setting, it will use the PHP setting for display_errors.

You can enable logging of errors by defining a class that implements the Interfaces\DebugLog interface and then nominating it Debug::SetLogStore( $classname )
